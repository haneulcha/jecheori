# DESIGN.md — 절기 스케치북

> UI를 만들거나 고치기 전에 반드시 읽는다. 모든 색·타이포·문구는 여기서 파생한다.
> 참조 스타일: Pa'lais (styles.refero.design, "botanical sketchbook dipped in honey")
> × 절기 달력. 하늘의 레퍼런스 선택으로 확정된 방향이다.

## 컨셉

손그림 요리책의 낱장이 계절 따라 넘어가는 앱. Pa'lais에게서 **질감**(크림 종이,
보태니컬 라인아트, 유기적 형태, 단일 잉크색의 규율)을, 절기 달력에게서
**시간**(절기 아이브로, 철마다 물드는 액센트)을 가져온다.

**시그니처 (과감함은 여기에만):** 쪽빛 라인아트 스케치가 놓인 머리말 —
절기가 적히고, 계절 액센트가 물든다. 나머지는 조용히.

## 색

**규율 (Pa'lais에게 배운 것): 텍스트·링크·버튼은 오직 쪽빛이 소유한다.**
계절 웜 컬러는 절대 글자를 싣지 않는다 — 배지 배경, 블롭, 칩 배경으로만 흐른다.
등락 마커(`--ink`/`--rise`)는 의미 전달로서 글자에 실린다.

공통 (계절 무관):

| 토큰 | 값 | 용도 |
|---|---|---|
| `--paper` | `#FBF9F6` | 배경. 크림 종이 (레퍼런스로 확정된 선택) |
| `--card` | `#FFFFFF` | 카드 — 크림 위에 순백 레이어링 |
| `--ink` | `#2B4586` | 쪽빛(藍). 모든 텍스트·링크·강조. 유일한 채도 있는 쿨톤 |
| `--muted` | `#7B84A3` | 보조 텍스트 (쪽빛의 뮤트) |
| `--line` | `#E7E2D6` | 구분선 (따뜻한 모래빛) |
| `--axis` | `#CBC2B0` | 스파크라인 x축 |
| `--margin` | `#E3C4B8` | 장보기 노트 여백선 |
| `--rise` | `#C0392B` | 러스트 / 가격 상승 마커(칩·큰가격) |
| `--rise-lo` | `#EFEAE3` | 상승 칩 저채도 배경 |

계절 액센트 (`body[data-season=…]` — 먹을 수 있는 웜 컬러만):

| 계절 | 월 | `--accent` | `--tint` | 심상 |
|---|---|---|---|---|
| spring | 3–5 | `#A2D3A6` | `#EAF4E9` | 새순 세이지 |
| summer | 6–8 | `#FFC400` | `#FFF4CE` | 꿀·참외 |
| autumn | 9–11 | `#ED7328` | `#FBE7D6` | 감·번트 오렌지 |
| winter | 12–2 | `#BC6E79` | `#F6E7EA` | 팥죽 |

`--accent`는 절정 배지 배경·블롭·카드 열림 테두리에, `--tint`는 칩과
머리말 블롭에. 글자는 그 위에서도 항상 쪽빛이다.

## 질감 — 보태니컬 라인아트

- 쪽빛 단색 스트로크의 잎가지 스케치(SVG, `currentColor`)를 머리말에 놓는다.
  토일(toile) 자수처럼 — 채우기 없이 선만, 낮은 불투명도(0.3~0.4)
- 머리말 뒤에 계절 `--tint` 색의 유기적 블롭 하나 (CSS border-radius 블롭 —
  자산 불필요)
- 일러스트는 과용하지 않는다: 머리말 1점, (선택) 빈 화면 1점. 카드 안엔 장식
  일러스트를 두지 않는다. 단 정보 그래픽(볼드 가격·등락 칩·가격 스파크라인·
  과일/채소 필터)은 장식이 아니라 데이터 전달이므로 카드에 허용된다.
- 열린 방향: 계절별 스케치 4종(봄 매화가지, 여름 열매가지, 가을 이삭, 겨울 마른가지)

## 타이포그래피

| 역할 | 폰트 | 사용처 |
|---|---|---|
| 본문·헤딩 | **Wanted Sans** (self-host woff2, 콘텐츠 서브셋, OFL) | 아이브로·h1·품목명·소제목·본문·가격 등 거의 전부. 웨이트 400/600/700/800 |
| 손글씨 | **학교안심 받아쓰기 L** (self-host woff2, 서브셋, OFL) | 카드 표지 하단 "제철이의 한마디"(`.why`) 한 줄만 — 유일한 캐릭터 폰트 |
| 라벨 | 본문 스택 + `letter-spacing: 0.08em` | "고르는 법"·"레시피" 같은 소형 라벨 |
| 숫자 | 본문 스택 + `font-variant-numeric: tabular-nums` | 가격 |

폰트 파일은 `src/fonts/`, CSS 상대 경로 참조 (Vite가 base 처리). CDN 금지.
못 구하면 시스템 고딕(`Apple SD Gothic Neo` …)으로 폴백.

**콘텐츠 서브셋:** 본문·손글씨 모두 전체 한글이 무거워, 콘텐츠(`produce.json`·`season.ts`·
`recipes.json`·`nutrition.json`·정적 UI 문구)에 실제 쓰인 글자만 서브셋한다(본문 4벌 ~240KB,
손글씨 ~30KB). 가격이 CI
JSON에서 오듯 폰트도 콘텐츠를 진실의 원천으로 삼는다. 문구를 고쳐 새 음절이 들어오면
`npm run subset:fonts`로 재생성한다 — 안 하면 `tests/font-coverage.test.ts`가 실패한다
(조용한 폴백 대신 시끄러운 실패).

## 절기

머리말 아이브로: `소서 · 7월 둘째 주`. 절기 날짜는 표기용 고정표
(`src/season.ts`) — 이 앱에서 절기는 시계가 아니라 계절 인사말이다.

## 레이아웃

- 단일 컬럼, 최대폭 28rem. 크림 캔버스 위 순백 카드. 카드 모서리는 종이처럼
  crisp하게 (`border-radius: 0.2rem`) — 둥근 타일이 아니라 메모 낱장.
- 구분은 여백과 1px 선. **그라데이션 금지.** 그림자는 "메모가 냉장고에 붙어 들뜬
  표현에 한해 아주 옅은 1단계(`--lift`, 카드에만)"만 예외. 그 외 그림자는 금지.
- 카드 펼침은 `<details>` — JS 없는 인터랙션 (과일/채소 필터도 라디오+CSS로 무JS)

> 구현 메모: 마크업은 `src/components/`의 React(JSX) 컴포넌트이고, 이 문서의 클래스명·
> 스타일은 `src/style.css`에서 그대로 재사용된다. 팔레트·타이포·규율은 프레임워크와 무관.

## 모션·접근성

- 모션은 절제해서 — 상태 전환을 부드럽게 하거나 시그니처를 살리는 데만. `prefers-reduced-motion: reduce`면 전부 끈다:
  - **카드 펼침**: `::details-content` 높이+투명도 리빌(표준 이징 `cubic-bezier(0.4,0,0.2,1)` 240ms,
    무JS·`interpolate-size`, Chrome 131+; 미지원은 즉시 펼침 폴백)
  - **필터칩·레시피칩**: 켜짐/꺼짐 색 전환 150ms
  - **레시피 메모**: 열기/닫기 대칭 1쌍(오버슈트 없음)
  - **차양(NavIndex)**: 그리드 높이 언롤 300ms(`cubic-bezier(0.2,0.75,0.2,1)`) + 램프줄 당김
  - **머리말 스케치**: 첫 로드 획 드로우인 0.7s(가지→잎→열매, 매 로드) — 시그니처 예외
- `:focus-visible`에 쪽빛 2px 아웃라인
- 웜 컬러 위 글자는 항상 쪽빛 — 명도 대비 4.5:1 이상을 색 규율이 보장한다

## 문구

CLAUDE.md의 규칙(한국어, 담백, 이커머스 화법 금지)에 더해:

- 계절의 목소리로: "7~8월이 절정이에요" ○ / "지금이 구매 적기!" ✕
- 감탄사·느낌표 금지. 서술로 끝낸다
- 실패·빈 화면도 같은 목소리로: "이번 달 제철 정보가 아직 없어요"

## 결정 기록

- 초안은 "절기 달력"(일러스트 없는 고요한 미니멀)이었다. 하늘이 Pa'lais
  레퍼런스를 가져와 논의 후 융합으로 확정 (2026-07-10)
- 크림 종이는 AI 기본값 팔레트와 겹치지만, 레퍼런스로 지정된 선택이므로 따른다.
  대신 쪽빛 단일 잉크 규율 + 계절 시스템 + 한글 부리체가 이 디자인의 고유성을 만든다
- 계절 가변 팔레트는 유지 — Pa'lais는 고정 브랜드지만, 이 앱의 주제는 계절이다
- 카드의 한마디(`whyNow`)를 펼침 영역 최상단에서 **표지 하단**으로 내리고 손글씨체로
  바꿨다 (2026-07-11). "제철이가 종이에 연필로 적어준 한 줄"이라는 심상 — 펼치기 전에도
  늘 보인다. 서명("— 제철이")은 넣지 않는다: 매 카드 반복되면 시끄럽고, 제철이는
  문구의 목소리로만 존재한다. 색은 규율대로 쪽빛의 뮤트(`--muted`), 웜 컬러 아님
- 카드를 냉장고 장보기 메모처럼 개편했다 (2026-07-11). 상단 마스킹테이프(반투명 계절색)+
  옅은 부양(`--lift`)+±0.4° 미세 기울기+crisp 모서리. 앱의 "장보기 메모" 은유를 눈에
  보이게. 그림자 금지 규율을 메모 고정 한정으로 완화(그라데이션 금지는 유지).
  스펙: `docs/superpowers/specs/2026-07-11-fridge-memo-card-design.md`
- 레시피 표시를 바텀시트에서 **카드 위 핀 메모 한 장**으로 바꿨다 (2026-07-12). 진입점은
  레시피별 칩(횡스크롤, `.chip-btn`), 상세는 압정으로 꽂힌 낱장(`.memo`) + `‹ ›` 넘김(clamp).
  냉장고-메모 은유의 연장. 그림자(메모 부양 1단)와 모션(열기/닫기 대칭 1쌍, 오버슈트 없음)에
  예외를 하나씩 더 열었다 — 마스킹테이프 완화와 같은 결. 압정은 단색(그라데이션 금지 유지).
  apple-design 스킬 렌즈로 다듬음(열기≠넘기기 분리, ~44px 터치 타깃, 칩 누름-즉시 피드백).
  스펙: `docs/superpowers/specs/2026-07-12-recipe-pin-memo-design.md`
- "다가오는 제철"을 메인 하단 한 줄에서 **별도 경로**(`/coming`)로 세우고, 두 페이지를
  **램프줄(A) + 차양 드롭 인덱스**(`NavIndex`)로 넘나들게 했다 (2026-07-12). 초안은 좌/우
  옆면 세로 인덱스 탭이었으나, 브라우저 실측에서 (1) 좌/우 배치가 페이지 넘김 연상을
  못 주고 (2) 모바일 단일 컬럼에서 고정 탭이 본문(품목 이모지)을 가려 폐기. 중간안이던
  풀하이트 슬라이드 드로워도 "화면을 덮는다"는 이유로 HTML 시안 비교 후 폐기하고, 우상단
  램프줄을 당기면 목차가 그 줄에서 아래로 풀려 내리는 차양(shade) 컴팩트로 확정 — 본문을
  가리지 않는다. 램프줄은 라인아트 어휘(선+끝점 원, 머리말 `Sprig`와 라임)를 그대로 쓰고
  iOS 아일랜드·유광 질감·햄버거 3줄(앱-크롬 클리셰)은 피했다. 줄을 뷰포트 위로 오버슈트시켜
  당겨도 위쪽에 틈이 생기지 않는다.
- 다가오는 페이지는 처음엔 냉장고-메모 카드의 표지 껍데기만 재활용한 얇은 `ComingCard`(가격·
  스파크라인·영양·레시피·펼침 없는 정적 `<div>`)였으나, 이후 메인과 **동일한 `ProduceCard`**로
  통합했다(`ComingCard` 폐기). 간트·펼침·손질법·영양·레시피는 메인과 같고, 가격만 작년 같은
  시기 씨앗(`coming-prices.json`)이라 등락 칩·스파크라인 없이 "작년 기준" 한 줄로만 보인다 —
  예고는 "볼 것"이지 "살 것"이 아니므로 등락 신호가 없는 게 자연스럽다. 마스킹테이프 색은 그
  **품목의 미래 달 계절색**(8월 품목은 여름 노랑, 9월 품목은 가을 오렌지)으로 물들여 "달력을
  미리 넘겨보는" 느낌을 색으로 얹었다. 이를 위해 계절 토큰 규칙을 `body[data-season]`(전역 한
  계절)에서 `[data-season]`(임의 요소)로 일반화 — `body`는 여전히 전역 기본을 주고, 카드마다
  그 위에 자기 계절을 재정의한다.
- 가격에 **기준선(단위) 줄**을 넣었다 (2026-07-14). `100g 기준` / `10개 기준 · 개당 704원`.
  단위가 없으면 감자 `315원`(100g)과 수박 `21,476원`(1개)이 같은 저울 위에 있는 것처럼
  읽힌다 — 단위는 장식이 아니라 숫자를 해석 가능하게 만드는 최소 조건이다.
  자리는 **큰 숫자 아래 muted 한 줄**(기존 개당값 자리를 확장). 개당값과 같은 계층인 게
  개념적으로 정확하다 — 둘 다 "이 숫자를 어떻게 읽나"를 말하는 각주다.
  시안 3개를 만들어 비교 후 확정. 탈락안과 이유:
  - **접두형**(`100g에 315원`, 한국어 어순대로 읽힘)은 자리를 만들려면 등락 칩을 취소선
    줄로 올려야 한다 → **머리기사인 "내려왔다"가 부제로 강등된다.** 대가가 너무 크다.
  - **라벨형**(취소선 줄 왼쪽에 작은 라벨)은 `100g  370원`이 한 줄에 붙어 라벨이 **취소선
    가격에 걸린 것처럼** 보인다. 명확히 하려다 새 모호함을 들인다. 라벨체(넓은 자간 소형)도
    냉장고 메모가 아니라 스프레드시트의 목소리다.
  엣지: 한 달 전 가격이 없으면(복숭아) 취소선·칩이 통째로 사라지는데 **기준선 줄은 남는다** —
  단위는 조건부가 아니다. 가격 자체가 없으면(KAMIS 미조사) 잴 것도 없으니 줄도 없다.
- 목차 링크는 평범한 `<a>` 대신 **TanStack `<Link viewTransition>`**(클라이언트 라우팅)로
  바꿨다. 초안의 `<a>`는 전체 페이지 하드 리로드라 "/"↔"/coming" 전환이 뚝 끊겼다(브라우저
  실측으로 확인). `Link`는 리로드 없이 `document.startViewTransition`으로 부드럽게 넘기고
  basepath도 자동 처리한다. 직접 URL 진입·무JS 폴백을 위해 CSS 크로스도큐먼트 view
  transition은 유지. `prefers-reduced-motion: reduce`면 차양 언롤·램프줄 당김·전환 전부 즉시로.
