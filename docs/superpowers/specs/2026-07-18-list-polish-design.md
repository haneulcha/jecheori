# 목록 시각 폴리시 — 설계

*`feat/full-seasonal-list`(PR #12) 위에 스택. 시안 사인오프 후 스펙(scratchpad `polish-siano.html`).
근거: [DESIGN.md](../../DESIGN.md), [제품-동작-지도](../../제품-동작-지도.md).*

작성: 2026-07-18. 전체 목록·검색·필터 라운드의 사용자 피드백 6건을 시각 폴리시로.

## 왜

전체 목록 라운드를 브라우저로 보고 사용자가 6건의 시각 피드백을 냈다. 데이터 변경(최근 최대 하락
기준선)은 별도 라운드로 떼고, 이번엔 **순수 시각/카피/레이아웃 폴리시**만 다룬다.

## 결정 (시안 사인오프 확정)

1. **조사일 줄 — 상대만, 절대날짜는 툴팁.** `5일 전 · 전국 평균`만 상시 표시하고, `5일 전`에
   호버(데스크톱)/탭(모바일)하면 `7월 13일 조사` 절대날짜가 툴팁으로. 점선 밑줄로 어포던스 표시.
   - **2026-07-16 "조사일 상시표시" 결정의 일부 되돌림**: 그 스펙은 상대+절대를 **항상** 인라인으로
     보였다("값싼 정직함"). 이번엔 절대날짜를 툴팁 뒤로 넣어 헤더를 간결히 한다. 절대날짜는 여전히
     **한 번의 상호작용으로 접근 가능** — 숨기는 게 아니라 접는 것. 결정기록에 남긴다.
   - 툴팁 상호작용은 기존 절정-점(`peak-dot`) 패턴 재활용(호버/포커스 + 탭 토글, 문서 위임).
2. **검색 플레이스홀더 한 단계 작게** — 입력값 폰트는 그대로, `::placeholder`만 `--text-md → --text-sm`.
3. **필터 칩 문구** — `내려간 것 → 가격 하락`, `절정 → 한창 제철`, `가격 있음`(유지). 과일/채소 유지.
   - "한창 제철": 페이지 카드가 이미 전부 제철이라 "제철" 단독은 무의미 → "한창"으로 절정을 또렷이.
   - "가격 하락": "내려간 것"의 모호함 해소(무엇이 내렸나 = 가격).
4. **정렬을 필터 칩과 같은 줄 우측 고정 + 칩 횡스크롤.** 칩 컨테이너는 `flex:1; overflow-x:auto`
   (스크롤바 숨김, 우측 페이드), 정렬 컨트롤은 `flex:0 0 auto`로 우측에 안 밀리고 고정.
5. **"정렬" 글자 → 아이콘 + 현재값.** 계단 막대 아이콘(내림차순 어휘)으로 라벨을 대체하고, 그 옆에
   현재 정렬값(`하락 큰 순`/`이름`/`가격 낮은 순`)을 유지 — 상태 은닉 방지(제품 UI는 현재값이 보여야).
   `<select>` 자체가 현재값을 표시하므로 셀렉트는 그대로 두고 "정렬" 텍스트만 아이콘으로 교체.
6. **카드 가격 재배치.** 취소선 예전가(`was`, 한 달 전 값) **제거**. 등락 칩을 큰 숫자 위로 올리고
   앞에 기간 라벨을 붙인다:
   ```
   지난달 대비  ↓26%        ← muted 라벨 + 등락 칩 (기존 chip 컴포넌트·화살표 재사용)
   5,352원                  ← 큰 숫자 (하락 쪽빛 / 상승 러스트)
   1kg 기준                 ← 단위 (그대로)
   ```
   - **비슷(±1%)**: 라벨 자리에 `지난달과 비슷` (칩 없음).
   - **기준선 없음**(한 달 전 값 결측, 예 복숭아): 라벨·칩 **없음** — 가격+단위만.
   - **"지난달 대비"는 데이터 라운드의 인계 지점.** 다음 라운드(최근 최대 하락 기준선)가 이 라벨을
     `지난주/2주 전/지난달 대비`로 **동적**으로 바꾼다. 이번엔 현재 데이터(monthAgo)에 맞춰 "지난달"
     고정. 레이아웃은 그때 안 바뀌고 라벨 텍스트·소스만 바뀐다.

## 로직 (조사일 라벨 분리)

`week.ts`의 `surveyedLabel`은 현재 `"오늘 · 7월 13일 기준"`을 한 문자열로 만든다. 이번엔 상대와 절대를
**나눠** 상대는 인라인, 절대는 툴팁에 넣어야 한다. 두 순수 헬퍼로 분리:

```ts
/** "오늘" | "5일 전" (표시용, 순수) */
export function relativeDayLabel(days: number): string {
  return days === 0 ? '오늘' : `${days}일 전`
}
/** "7월 13일 조사" (표시용, 순수) — 툴팁용 절대날짜 */
export function surveyedDateLabel(surveyedOn: string): string {
  const [, m, d] = surveyedOn.split('-')
  return `${Number(m)}월 ${Number(d)}일 조사`
}
```

기존 `surveyedLabel`은 더 쓰이지 않으면 제거(App만 소비). App은 `relativeDayLabel(days)`를 점선
밑줄 span으로 렌더하고 그 안에 `surveyedDateLabel(surveyedOn)`을 툴팁으로, 뒤에 `· 전국 평균`.
