# 냉장고 메모 카드 설계

> 2026-07-11 · 브레인스토밍 산출물
> 관련: `DESIGN.md`(팔레트·타이포·규율), `docs/superpowers/specs/2026-07-10-card-info-and-price-display-design.md`(카드 구조)

## 배경

앱의 정체성은 이커머스가 아니라 "장보기 갈 때 옆에 두는 계절 메모"다(`CLAUDE.md`).
현재 카드는 크림 배경 위 순백 카드(1px 테두리·라운드)로 정갈하지만, 그 "메모" 은유가
시각적으로 드러나지 않는다. 카드를 **냉장고에 마스킹테이프로 붙인 장보기 메모 한 장**처럼
보이게 해 은유를 눈에 보이게 한다. 톤(담백·손그림·쪽빛 규율)은 유지한다.

이미 카드 표지 하단에 손글씨 한마디(`.why`, 학교안심 받아쓰기 L)가 있어 "메모" 정체성의
절반은 완성돼 있다. 이 설계는 **고정 모티프(테이프) + 아주 옅은 부양(그림자) + 미세 기울기**를
더해 나머지 절반을 채운다.

## 결정 (브레인스토밍)

한 번에 하나씩 좁힌 결과:

1. **메모 상(像):** 포스트잇도 찢은 쪽지도 아닌 **장보기 메모패드 한 장**.
2. **들뜸:** 평평 유지가 아니라 **아주 살짝 들뜬**(옅은 그림자 1단계) — 규율 예외.
3. **고정 모티프:** 자석/코너테이프가 아닌 **마스킹테이프 한 조각**(상단 중앙).
4. **노트 결:** 여백선·괘선 없이 **깔끔하게**(테이프+그림자+손글씨로만 메모 표현).
5. **기울기:** **아주 살짝·교대로**(±0.4°) — 손으로 붙인 느낌, 가독성 유지.
6. **종이 톤:** **순백 유지**(크림 배경 위 백지 레이어링 그대로).

## 원칙 (유지)

- 런타임 의존성 0, 런타임 외부요청 0. 새 에셋 없음. 인터랙션은 CSS 우선(JS 없음).
- 마크업은 거의 그대로. 시각 변화는 전부 `src/style.css`에서.
- 웜 컬러는 배경 도형으로만, 글자·강조는 쪽빛. 테이프는 배경 도형이라 계절색 허용.
- 카드 안 일러스트는 여전히 없음. 테이프·그림자는 장식이지만 **머리말 스케치처럼 절제된
  단일 시그니처**로 취급하고, 그 외는 조용히 유지한다.

## 규율 개정 1건 (DESIGN.md)

기존: "구분은 여백과 1px 선. **그라데이션·그림자 금지**(블롭과 스케치가 유일한 장식)."

개정: **그라데이션 금지는 유지.** 그림자는 "메모가 냉장고에 붙어 들뜬 표현에 한해 아주 옅은
1단계"를 예외 허용한다. 남용 방지를 위해 값을 토큰(`--lift`)으로 고정하고 카드에만 쓴다.

## 요소 명세

전부 CSS. 마크업 변경 없음(테이프·그림자는 `.card`의 pseudo-element·속성).

### 1. 마스킹테이프 — `.card::before` (시그니처)

- 위치: 상단 중앙, `top: -0.6rem`으로 카드 위 경계에 걸쳐 "붙인" 느낌.
- 크기: 폭 ~4.5rem, 높이 ~1.3rem.
- 색: 반투명 계절색 `color-mix(in srgb, var(--accent) 40%, transparent)`.
  **그라데이션 아님** — 단일 반투명 평면(마스킹테이프의 반투명감으로 충분).
- 회전: `rotate(-2deg)` (테이프 한 조각의 붙인 각도).
- 접근성: 순수 장식. pseudo-element라 접근성 트리에서 자동 제외.

### 2. 들뜬 그림자 — `.card`

- `--lift: 0 2px 6px rgba(43, 69, 134, 0.06)` 토큰(쪽빛 살짝 섞인 매우 옅은 부양).
- `.card { box-shadow: var(--lift); }` — 카드에만. 다른 요소에 그림자 금지 유지.
- 정적 값이라 `prefers-reduced-motion` 무관.

### 3. 미세 기울기 — `.card:nth-child(odd|even)`

- 홀수 `rotate(0.4deg)`, 짝수 `rotate(-0.4deg)` 교대.
- 회전 컨테이닝 블록이 생겨 `.card::before`(absolute)의 기준이 카드로 잡힘 — 의도된 동작.
- 아주 작은 각도라 28rem 컬럼의 수평 넘침은 무시 가능(컬럼 좌우 패딩으로 흡수).

### 4. 카드 간격

- 테이프가 위 카드에 닿지 않도록 `.card` 세로 간격을 `margin-bottom: 0.7rem → ~1.4rem`.

### 5. 종이처럼 crisp한 모서리

- `.card` 라운드를 `0.9rem → 0.2rem`로 줄여 메모 종이 한 장처럼. 둥근 카드보다 종이 낱장에 가깝다.
- 칩·블롭·필터의 둥근 형태와 대비되어 카드가 "종이"로 읽힘. DESIGN.md 레이아웃의
  `border-radius: 0.9rem` 기재를 함께 개정한다.

### 6. 유지

- 순백 카드, 1px 모래빛 테두리, 크림 배경, 계절 시스템, 머리말 스케치,
  손글씨 한마디(`.why`) — 그대로.

## 견고성·접근성

- 테이프는 텍스트 위에 올라가지 않음(상단 여백에 위치) → 명도 대비 영향 없음.
- 계절 전환 시 테이프 색은 `--accent`를 따라 자동으로 물듦(봄 세이지·여름 꿀·가을 감·겨울 팥).
- `color-mix` 미지원 구형 브라우저: 테이프가 불투명 `--accent`로 폴백돼도 무해(살짝 진할 뿐).
- 회전·그림자는 정적. 모션 없음.

## 범위 밖 (YAGNI)

- 테이프 끝 페이드(그라데이션/마스크), 종이 질감 텍스처, 자석/코너테이프 변형, 괘선.
- 카드 펼침 시 테이프 애니메이션. 지금은 정적 한 장으로 충분.

## 구현 위치

- `src/style.css` — `.card` 그림자·간격·기울기, `.card::before` 테이프, `:root`의 `--lift` 토큰.
- `DESIGN.md` — 그림자 규율 개정 + 결정 기록 한 줄.
- 마크업(`src/components/ProduceCard.tsx`)·로직·테스트 변경 없음(시각 전용).
- 브라우저 실측 검증(테이프 위치·그림자·기울기·계절색·수평 넘침 없음) 후 완료.
